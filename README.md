# 일정관리 앱 과제
## 프로젝트 소개
일정을 등록하고 수정, 삭제할 수 있는 스케쥴러 구현

## 개발 기간
2024.10.01 ~ 2024.10.04

## 개발 환경
- spring boot 3.3.4
- IntelliJ

## 주요 기능
- 첫 화면에서 일정 등록 버튼을 누르면 일정을 입력하고 등록
- 일정 선택 후 선택 일정 수정, 삭제
